# llm

This repository provides a basic wrapper around Llama v2 model using LlamaCpp to create a chatbot using a gradio chat interface that supports streaming for quick responses.

* To download a model, refer to https://github.com/ggerganov/llama.cpp#obtaining-and-using-the-facebook-llama-2-model
* For instructions on setting up Llamacpp, refer to https://python.langchain.com/docs/guides/local_llms
* For a guide on creating chatbots using Gradio, refer to https://www.gradio.app/guides/creating-a-chatbot-fast
